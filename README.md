# Jogo de Adivinhação em JavaScript

Bem-vindo ao Jogo de Adivinhação em JavaScript! Este é um programa simples onde você tenta adivinhar o número que o computador pensou.

## Como Jogar

1. Clone ou baixe este repositório para o seu computador.
2. Abra o arquivo `index.html` no seu navegador.

### Regras do Jogo

- O computador pensará em um número entre 1 e 10.
- Você tentará adivinhar o número digitando sua resposta em um campo de entrada.
- O programa fornecerá dicas, indicando se sua resposta é muito alta, muito baixa ou correta.
- Continue a adivinhar até acertar o número.

## Executando Localmente

Você pode simplesmente abrir o arquivo `index.html` em seu navegador ou usar um servidor web local se preferir.

### Usando Python

Se você tiver o Python instalado, pode usar o seguinte comando no terminal para iniciar um servidor web local:

```bash
python -m http.server
```

Depois, acesse `http://localhost:8000` no seu navegador.
